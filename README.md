# ToolShare – UiTM Student Resource Sharing App

ToolShare is a mobile application developed for **UiTM Shah Alam students** to share, rent, and swap tools, equipment, and academic resources.  
The app aims to reduce student costs, encourage sustainability, and provide a secure platform for peer-to-peer item exchange.

---

##  Key Features
-  **User Authentication (Firebase)** – Secure login and registration  
-  **Item Listing** – Upload, view, edit, and delete items  
-  **Filters & Search** – Filter by All Items, My Items, Borrowed Items, etc.  
-  **QR Code System** – Borrow and return items using QR scan  
-  **GPS & Map Integration** – Locate items around UiTM Shah Alam  
-  **Real-Time Notifications** – Borrow requests & return reminders  
-  **Status Tracking** – Track borrowed and lent-out items  

---
